Accordion Menu

It's a jQuery accordion menu plugin, and I call it acc3menu!

acc3menu allows you to create a tree level accordion menu

  • Comsumes less space
  • Damn sexy
  • Not another dull form (y'know, something different, breaking the habit)

Your form requires little modification, so even if JavaScript is turned off or Promin fails to deliver, your form won't break.
I've tried several browsers on desktops, tablets and phones. So far, everything seems to work like a charm.

How does one implement?

It all starts with a single line of JavaScript code...

$( form ).acc3menu( options );

... and the HTML code

<ul id="menu">
    <li>
        <a href="#" class="">link 1 Level 1</a>
        <ul>
            <li><a href="#">subLink1</a></li>
            <li><a href="#">subLink2</a></li>
            <li><a href="#">subLink3</a></li>
        </ul>
    </li>
    <li>
        <a href="#" class="">link 2 Level 1</a>
        <ul>
            <li><a href="#">subLink1</a></li>
        </ul>
    </li>
    <li>
        <a href="#" class="">link 3 Level 1</a>
        <ul>
            <li>
                <a href="#">subLink1</a>
                <ul>
                    <li><a href="#">subSubLink1</a></li>
                    <li><a href="#">subSubLink2</a></li>
                    <li><a href="#">subSubLink3</a></li>
                </ul>
            </li>
        </ul>
    </li>
</ul>
